# 黄蓉 - 小红书运营工作清单

## 🎯 岗位定位
- **角色**: 电竞国度小红书首席运营
- **账号**: 电竞国度 (CyberNation)
- **核心目标**: 建立品牌桌面美学心智，获取高意向装机/外设线索。

## 📋 日常任务 (Daily)
- [ ] **内容发布**: 每日 08:00 - 10:00 发布 1 篇笔记（图文/视频）。
- [ ] **数据统计**: 每日 08:00 汇总前日笔记流量、点赞、收藏、评论。
- [ ] **评论回复**: 实时监控评论区，回复技术咨询，引导私信。

## 📅 周期任务
- **每周复盘 (Weekly)**: 每周日 20:00 选题库更新，分析本周爆款原因。
- [ ] **每月报告 (Monthly)**: 每月 1 号生成月度运营分析报告及下月计划。

## ⚙️ 自动化配置
- **Cron 任务**: `0 8 * * *` (每日 8 点执行数据抓取与统计)
- **心跳检查**: 每 2 小时执行一次评论区关键词扫描。

## 🚨 特殊情况处理
- **审核未通过**: 立即检查敏感词，修改后重新提交。
- **负面评论**: 2 小时内冷处理或专业回复，防止舆论发酵。
- **爆款处理**: 流量过万时，增加评论区互动频率，加强私信转化。
- **私信处理**: 15 分钟内响应，引导至企业微信。
